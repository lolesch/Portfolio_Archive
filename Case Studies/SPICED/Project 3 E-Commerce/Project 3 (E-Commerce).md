#### Week 1
- we started with a user story, defined user needs and decided on a product.
- From here we created 3 design principles that guided our process
- A moodboard supported this early exploration and visual direction
- we defined a user flow and scetched our first wireframes
- A style guide was established for color, typography, Iconography and tested in one of the screens.
- On this basis we created a list of UI Elements for our Design System using atomic design, converted them into components and doublechecked them against our principles.

#### Week 2
- we created mid-fidelity designs and started prototyping.
- Here we started using variables and created a proof of concept for the editor to see if we could manage it in the long run.
- micro-interactions were added

#### Week 3
- creating and conducting usability testing
- the challenges and learnings from that
- the results/findings and the affinity/prioritization matrix
- planed next steps
- iterations, mainly on the editor side panel
- presentation and final prototype

### Key Challenge

Finding the right interaction between variables, modes and where to apply them in nested instance variants costed a lot of time when it came to troubleshooting.
Also micro interactions caused problems with not correctly applying variables due to nested interaction priority.
Another think was the amount of work to set everything up correctly while the project was still going through major design changes. this lead to unnecessary loss of state that was working beforehand.

## Learnings
- [ ] what are the takeaways from this learning experience?
- [ ] list all the learnings from variable usage and usability testing

- no support for smart animating variable mode changes.
- issues with 'scroll to' setups when using instance variants
- agile usability testing worked best for us due to fast iteration cycles
- do not rush wireframing
	- this was a flaw from the project setup, where we needed to get to a testable state ASAP
- design-principles and -system are very powerful when applied early on and consistently
